# usage 

1. cd caffe
2. mkdir build
3. cmake ..
4. make all
5. cd lib
6. make

the model.prototxt is in

```
models\pascal_voc\VGG16\FP_Net_end2end\train.prototxt
```

train:
```
./experiments/scripts/FP_Net_end2end.sh 1 VGG16 pascal_voc
```
test:
```
./test.sh 1 VGG16 pascal_voc

```
1 is your gpu index
